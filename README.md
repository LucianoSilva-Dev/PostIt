# Dream Board (Post-it)

## Visão Geral
O Dream Board é uma aplicação web interativa projetada para capturar e exibir ideias ou sonhos de empreendedorismo de forma dinâmica e visualmente atrativa. A ideia central é permitir que os usuários insiram suas ideias em formato de "post-its" através de uma interface simples, e as mensagens sejam exibidas de forma criativa em uma tela que simula um quadro negro de sala de aula.

## Funcionalidades
A aplicação é dividida em duas páginas principais:
1. **Página de Submissão:** Uma interface intuitiva onde o usuário pode digitar seu sonho ou ideia, e escolher a cor do texto do seu post-it.
2. **Página de Exibição (Projetor):** Uma tela projetada para exibição em tela cheia (como em um projetor). Ela possui o visual de um quadro negro escolar, onde os post-its aparecem de forma animada (flutuantes, com zoom-in/zoom-out e rotacionados aleatoriamente). Um diferencial técnico importante é o sistema de detecção de colisão que garante que o **texto** de um post-it não sobreponha o texto de outro (a parte em branco do post-it pode se sobrepor para dar um efeito realista, mas o texto precisa estar legível).

## Tecnologias Utilizadas
- **Frontend:** HTML5, CSS3 puro (sem Tailwind), e JavaScript (Vanilla).
- **Backend:** Node.js com framework Express.
- **Banco de Dados:** PostgreSQL com Prisma ORM.
